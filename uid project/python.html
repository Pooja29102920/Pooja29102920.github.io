1. Introduction to Python

1.1 What is Python?


Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • web development (server-side),

  • software development,

  • mathematics,

  • system scripting.


Python is a high-level, interpreted programming language known for its simplicity and readability...

2. Variables and Data Types

2.1 Understanding Variables


Variables are containers for storing data values. In Python, variables are created when you assign a value to it.

Key points about Python variables:

  • No declaration needed

  • Case-sensitive

  • Can change type after creation

  • Must start with a letter or underscore



    # Creating variables
    x = 5           # Integer
    name = "John"   # String
    price = 19.99   # Float
    is_valid = True # Boolean
    
    # Print variables
    print(x)
    print(name)
    print(price)
    print(is_valid)

2.2 Data Types in Python


Built-in Data Types in Python include:

  • Text Type: str

  • Numeric Types: int, float, complex

  • Sequence Types: list, tuple, range

  • Mapping Type: dict

  • Boolean Type: bool



    # Examples of different data types
    text = "Hello"          # str
    number = 20            # int
    decimal = 20.5         # float
    complex_num = 1j       # complex
    my_list = [1, 2, 3]    # list
    my_tuple = (1, 2, 3)   # tuple
    my_dict = {"name": "John", "age": 30}  # dict
    is_true = True         # bool
    
    # Check type of variable
    print(type(text))
    print(type(number))
    print(type(decimal))

2.3 Type Conversion


Type Conversion is the process of converting data from one type to another.

Common type conversions:

  • str() - Convert to string

  • int() - Convert to integer

  • float() - Convert to float

  • bool() - Convert to boolean



    # Type conversion examples
    x = str(3)      # '3'
    y = int("3")    # 3
    z = float("3.14") # 3.14
    b = bool(1)     # True
    
    print(x, type(x))
    print(y, type(y))
    print(z, type(z))
    print(b, type(b))
