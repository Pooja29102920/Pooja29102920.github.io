1. Introduction to JavaScript

1.1 What is JavaScript?


JavaScript is a dynamic programming language primarily used for web development. It was created by Brendan Eich in 1995.

It is used for:

  • Web Development (Client-side)

  • Server-side Development (Node.js)

  • Mobile App Development

  • Game Development


JavaScript makes web pages interactive and dynamic...

console.log("Hello, World!");
// This is your first JavaScript program

1.2 Setting Up JavaScript Environment

  • Modern web browser (Chrome, Firefox, etc.)
  • Text editor (VS Code, Sublime Text)
  • Node.js installation (optional)
  • Browser Developer Tools

2. Variables and Data Types

2.1 Understanding Variables


Variables in JavaScript are containers for storing data values. They can be declared using var, let, or const.

Key points about JavaScript variables:

  • Use let for variables that can change

  • Use const for constants

  • Case-sensitive naming

  • Dynamic typing supported


// Variable declarations
let name = "John";    // String
const age = 25;       // Number
let isStudent = true; // Boolean
var price = 19.99;    // Number (float)

// Printing variables
console.log(name);
console.log(age);
console.log(isStudent);

2.2 Data Types in JavaScript


Basic Data Types in JavaScript include:

  • String: Text values

  • Number: Integers and floating-point

  • Boolean: true/false

  • Object: Collections of data

  • Array: Ordered lists

  • Undefined: Unassigned values


// Data type examples
let text = "Hello";           // String
let number = 42;              // Number
let decimal = 3.14;           // Number
let active = true;            // Boolean
let colors = ["red", "blue"]; // Array
let person = {               // Object
    name: "John",
    age: 30
};

// Check types
console.log(typeof text);
console.log(typeof number);
console.log(typeof person);
